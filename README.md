# Employee Management System

This is a backend project built using Spring Boot and MySQL.

## Features

* Create Employee
* Get All Employees
* Get Employee by ID
* Update Employee
* Delete Employee

## Tech Stack

* Java
* Spring Boot
* MySQL
* REST API

## API Endpoints

| Method | Endpoint            |
| ------ | ------------------- |
| POST   | /api/employees      |
| GET    | /api/employees      |
| GET    | /api/employees/{id} |
| PUT    | /api/employees/{id} |
| DELETE | /api/employees/{id} |

## How to Run

1. Clone the repository
2. Configure MySQL in application.properties
3. Run using:
   mvnw spring-boot:run
